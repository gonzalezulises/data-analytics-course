# Introducción a Dashboards

Los dashboards transforman análisis en productos interactivos. Comunican insights de forma visual y accesible.

## Herramientas principales

| Herramienta | Fortaleza |
|-------------|-----------|
| Plotly | Gráficos interactivos |
| Streamlit | Apps rápidas en Python |
| Dash | Dashboards complejos |
| Panel | Notebooks a apps |

## El stack que usaremos

- **Plotly Express**: Gráficos interactivos simples
- **Plotly Graph Objects**: Control total
- **Streamlit**: Framework para apps

## ¿Por qué Streamlit?

- 100% Python
- Sin frontend (HTML/CSS/JS)
- Despliegue simple
- Ideal para prototipos y MVPs
